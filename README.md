# Python Mini-Projects (ITGN130)

This repository contains two functional Python applications developed as part of the Summer I 2023-24 academic year.

## 1. Tic-Tac-Toe Game
A classic two-player game played on a 3x3 grid.
- **Run with**: `python tic_tac_toe.py`
- **Goal**: Connect three marks (X or O) horizontally, vertically, or diagonally.

## 2. QR Code Generator & Decoder
A command-line tool to create and read QR codes using external libraries.
- **Run with**: `python qr_code_app.py`
- **Dependencies**: Requires `qrcode` and `Pillow` (see requirements.txt).
